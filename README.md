# 🌤️ Weather Forecast Application

A modern and responsive Weather Forecast Web Application built using HTML, CSS, and Vanilla JavaScript.

This project uses a local JavaScript dataset instead of a weather API, making it fully offline and beginner-friendly.

---

## 🚀 Features

- Search Weather by City
- Current Weather Details
- 7-Day Weather Forecast
- Loading Indicator
- Local Storage Support
- Search History
- Live Date & Time
- Temperature Unit Toggle
- Responsive Design
- Glassmorphism UI
- Automatic Weather Update Every Minute (Simulated)
- Offline Project (No API)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
